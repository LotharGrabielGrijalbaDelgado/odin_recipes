🍽️ Sitio de Recetas
📌 Descripción del proyecto

Este proyecto fue desarrollado como parte del proceso de aprendizaje de un estudiante de The Odin Project. Su objetivo principal es servir como un sitio de práctica donde se aplican y consolidan habilidades fundamentales de desarrollo web, especialmente en HTML.

El sitio presenta una colección de recetas organizadas de forma sencilla, permitiendo al usuario navegar entre ellas y visualizar sus ingredientes y pasos de preparación.

🎯 Objetivos
Practicar la estructura básica de documentos HTML
Aplicar buenas prácticas de maquetación semántica
Crear navegación entre múltiples páginas
Organizar contenido de forma clara y accesible
🛠️ Tecnologías utilizadas
HTML5
📚 Lo que se puso en práctica

Durante el desarrollo de este proyecto se trabajaron conceptos clave como:

Uso de etiquetas semánticas (header, main, footer, etc.)
Creación de listas ordenadas y desordenadas
Inserción de imágenes y enlaces
Estructuración de contenido en múltiples páginas
🚀 Estado del proyecto

Este proyecto es una versión inicial enfocada en el aprendizaje. Puede ser mejorado en el futuro mediante la incorporación de:

CSS para mejorar el diseño visual
JavaScript para añadir interactividad
Diseño responsive para distintos dispositivos
📖 Aprendizajes

Este proyecto permitió reforzar los fundamentos del desarrollo web y comprender cómo estructurar correctamente un sitio desde cero
